HTML 基礎教學

HTML（超文本標記語言）是構建網站的基礎。它用來描述網頁的結構。

基本標記

一些基本的 HTML 標記包括：

  • <html>: 定義 HTML 文件。
  • <head>: 網頁的頭部資訊。
  • <body>: 網頁的主內容。

範例

            
            <html>
                <head>
                    <title>範例標題</title>
                </head>
                <body>
                    <h1>歡迎來到我的網站</h1>
                </body>
            </html>
